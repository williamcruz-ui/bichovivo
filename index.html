Capacidades

O que BichoVivo oferece

👀

Rastreio de Olhos

O bichinho segue o movimento do seu cursor. Quanto mais interage, mais conectado fica.

🔊

Sons Sintetizados

Web Audio API gera sons únicos em tempo real. Cada ação tem seu próprio som característico.

😄

Humor Próprio

4 temperamentos diferentes: Brincalhão, Tranquilo, Resmungão ou Carente. Cada um reage diferente.

📊

Sistema de Stats

Gerenciar saciedade, felicidade, energia, higiene e saúde. Dados persistem com localStorage.

😴

Ciclo Sono/Vigília

O bichinho dorme quando cansado e recobra energia. Respeita seus horários naturais.

📡

Radar Social

Veja outros bichões próximos. Sistema de detecção de proximidade simulado com pool de personagens.

Tecnologia

Como foi feito

✓ React 18 — Componentes modernos com hooks
✓ Web Audio API — Síntese de sons
✓ localStorage — Persistência de dados
✓ Rastreio de movimento — Interatividade em tempo real
✓ SVG animations — Gráficos vetoriais suaves

BichoVivo demonstra técnicas modernas de desenvolvimento web com foco em interatividade, design responsivo e experiência do usuário.

Cada elemento foi cuidadosamente desenhado pra criar uma conexão emocional entre você e seu bichinho digital.
